# nginx_monitoring
Zabbix 4 LTS Nginx monitoring by Alireza Zabihi based on Alex Gluck solution on Centos 7.6


Connection Statistics:
- Active
- Reading
- Waiting
- Writing
- Requests pro connection
- Request Statistics:
- Accepted
- Handled
- Total

Linux and Nginx performance monitoring:
- File descriptors (Maximum and Current)
- CPU Cores
- Worker_Connections
- Worker_Processes
- Maximum supported connection depended on configuration
- Nginx service status

Graph and screen:
- 'Requests Statistics' graph
- 'Connection Status' graph
- File descriptors

Triggers:
- Nginx service status
- Worker processes bad configuration
